<Admonition type="warning">
The `system` object is **only available in the Main script**. UI scripts should communicate with the main script via [IPC](/api/widget-api/widget-methods/#inter-process-communication-ipc) if they need system data.
</Admonition>

The Audio API provides methods to control the master system volume and play WAV audio files.

---

## **`system.getVolume()`**

Retrieves the current master system volume.
---

### Return Value

* **Type**: `number`
* **Description**: An integer between `0` and `100` representing the current volume percentage.

---

<Method signature="system.setVolume(level)"
  description="Sets the master system volume.">
  <MethodParam name="level" type="number">
    An integer between `0` and `100` representing the target volume percentage. Values outside this range will be clamped.
  </MethodParam>
</Method>

<Method signature="system.playSound(path, [loop])"
  description="Plays a WAV audio file asynchronously.">
  <MethodParam name="path" type="string">
    Path to the WAV file. Can be absolute or relative to the widget script.
  </MethodParam>
  <MethodParam name="loop" type="boolean">
    **Default**: `false`

    If `true`, the sound will loop continuously until `stopSound()` is called.
  </MethodParam>
</Method>

### Return Value

* **Type**: `boolean`
* **Description**: Returns `true` if the sound playback was successfully initiated.

---

## **`system.stopSound()`**

Stops any currently playing sound initiated via `playSound()`.
---

### Examples

#### 1. Volume Control
<CodeBlock language="javascript">
  {`// Get current volume
var vol = system.getVolume();
console.log("Current Volume: " + vol);

// Set volume to 50%
system.setVolume(50);`}
</CodeBlock>

#### 2. Playing an Alert Sound
<CodeBlock language="javascript">
  {`// Play a sound once
system.playSound("assets/alert.wav");

// Play a background sound on loop
system.playSound("assets/ambient.wav", true);

// Stop the sound after 10 seconds
setTimeout(function() {
    system.stopSound();
}, 10000);`}
</CodeBlock>
