<Admonition type="warning">
The `system` object is **only available in the Main script**. UI scripts should communicate with the main script via [IPC](/api/widget-api/widget-methods/#inter-process-communication-ipc) if they need system data.
</Admonition>

The Clipboard API allows widgets to interact with the system clipboard.

## Methods

<Method signature="system.getClipboardText()"
  description="Retrieves the current text content of the clipboard.">
    <ReturnValue type="string|null">
     The text content of the clipboard, or `null` if the clipboard is empty or does not contain text.
  </ReturnValue>
</Method>

<Method signature="system.setClipboardText(text)"
  description="Sets the text content of the system clipboard.">
  ### Parameters
  <MethodParam name="text" type="string">
    The text to copy to the clipboard.
  </MethodParam>
  ### Return
  <ReturnValue type="boolean">
    `true` if successful.
  </ReturnValue>
</Method>

## Examples

<CodeBlock language="javascript">
  {`// Copy text to clipboard
var success = system.setClipboardText("Copied from Novadesk!");
if (success) {
    console.log("Text copied!");
}

// Read from clipboard
var text = system.getClipboardText();
console.log("Clipboard contains: " + text);`}
</CodeBlock>
