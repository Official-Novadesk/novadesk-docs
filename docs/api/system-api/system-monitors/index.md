# System Monitors

System monitors provide real-time monitoring capabilities for various system resources and metrics.

## Available Monitors

- [CPU Monitor](./cpu-monitor) - Monitor CPU usage and performance metrics
- [Disk Monitor](./disk-monitor) - Track disk usage and I/O operations
- [Memory Monitor](./memory-monitor) - Monitor RAM usage and memory statistics
- [Mouse Monitor](./mouse-monitor) - Track mouse activity and movements
- [Network Monitor](./network-monitor) - Monitor network traffic and connectivity

Each monitor provides specific APIs for accessing real-time system information and can be used to create dynamic widgets that respond to system changes.
