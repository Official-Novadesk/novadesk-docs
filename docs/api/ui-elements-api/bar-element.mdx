## Adding an Bar Element

Use the `addBar()` method on the `win` object within a [UI Script](/guides/script-types/#ui-script-the-face):

<CodeBlock language="javascript">
    {`win.addBar(options);`}
</CodeBlock>

## Options

The bar element inherits all [General Element Options](/api/ui-elements-api/general-elements-options/) and adds the following:

<Prop name="value" type="number" defaultValue="0.0">
  The fill level of the bar, from `0.0` (empty) to `1.0` (full).
</Prop>

<Prop name="orientation" type="string" defaultValue="horizontal">
  The direction in which the bar fills.
  - `horizontal`: Fills from left to right
  - `vertical`: Fills from bottom to top
</Prop>

<Prop name="barCornerRadius" type="number" defaultValue="0">
  Corner radius applied to the bar fill.
</Prop>

<Prop name="barColor" type="string">
  The [color](/guides/colors/) of the filled portion of the bar.
</Prop>

<Prop name="barColor2" type="string">
  The [color](/guides/colors/) used to create a gradient fill.
</Prop>

<Prop name="barGradientAngle" type="number" defaultValue="0.0">
  Gradient angle (in degrees) used when `barColor2` is set. Supports the full 360? range for precise control over the fill direction.
</Prop>

## Example

<Tabs>
    <TabItem value="index-js" label="index.js" default>
        <CodeBlock language="javascript">

            {`var exampleBarWindow = new widgetWindow({
    id: "exampleBar",
    width: 300,
    height: 40,
    backgroundColor:"rgba(255, 255, 255, 1)",
    script: "ui/ui.js",
});`}
        </CodeBlock>
    </TabItem>

    <TabItem value="ui-ui-js" label="ui/ui.js">
        <CodeBlock language="javascript">
            {`win.addBar({
    id: "cpu-Bar",
    x: 50,
    y: 10,
    width: 200,
    height: 20,
    value: 0.5,
    barColor: "rgba(20, 232, 115, 1)",
    solidColor: "rgba(241, 216, 19, 1)",
    solidColorRadius: 8,
    barCornerRadius: 8
})  `}
        </CodeBlock>
    </TabItem>
</Tabs>
