## Options

<Prop name="id" type="string">
  **Required**: `true`  
  Unique identifier for the elements. If an element with the same ID already exists, it will be **replaced**.
</Prop>

<Prop name="x,y" type="number" defaultValue="0">
  Specifies the x (horizontal) and y (vertical) position of the element in pixels relative to the top-left edge of the widget.
</Prop>

<Prop name="width,height" type="number" defaultValue="auto calculate">
  Specifies the width and height of the element in pixels.
</Prop>

<Prop name="solidColor, solidColor2" type="string" defaultValue='""'>
  [color](/guides/colors/) of the element background. If `solidColor2` is also specified, the background is a gradient composed of
  `solidColor` and `solidColor2`.
</Prop>

<Prop name="gradientAngle" type="number" defaultValue="0">
  Angle of the background gradient in degrees when both `solidColor` and `solidColor2` are defined. This accepts a raw number.
</Prop>

<Prop name="solidColorRadius" type="number" defaultValue="0">
  Corner radius for solid color background.
</Prop>

<Prop name="rotate" type="number" defaultValue="0">
  Rotation angle in degrees.
</Prop>

<Prop name="antiAlias" type="bool" defaultValue="true">
  If set to `true`, antialising (edge smoothing) is used when the element is drawn.
</Prop>

<Prop name="bevelType" type="string" defaultValue="none">
  Draws a bevel around the edges of the rectangle defined by `width` and `height`.  
Controls the visual style of the bevel.

* **Valid values**:
  - **`"none"`** : No bevel
  - **`"raised"`** : Raised bevel effect
  - **`"sunken"`** : Sunken bevel effect
  - **`"emboss"`** : Embossed effect (3D border)
  - **`"pillow"`** : Pillow effect (faded inside)
</Prop>

<Prop name="bevelWidth" type="number" defaultValue="1">
  Sets the width of the bevel in pixels. Optional; defaults to `1` when a `bevelType` is specified.
</Prop>

<Prop name="bevelColor, bevelColor2" type="color string">
  **Default**:
  - **`bevelColor`**: `"255,255,255,200"`
  - **`bevelColor2`**: `"0,0,0,150"`

  Used when `bevelType` is enabled. Controls the colors applied to the bevel edges.

  **Behavior**:
  - For **`raised`** and **`sunken`** bevel types, these colors create highlight and shadow effects.
  - **`bevelColor`** typically acts as the primary (highlight) color.
  - **`bevelColor2`** provides the contrasting (shadow) color.
</Prop>

<Prop name="padding" type="number | array" defaultValue="0">
  Adds padding (in pixels) around the content of the element. The padding area is drawn using the colors defined by `solidColor` and `solidColor2`.

* **Syntax**:
  - `padding: 10` : All sides set to 10px
  - `padding: [horizontal, vertical]` : e.g. `[5, 10]`
  - `padding: [left, top, right, bottom]` : e.g. `[5, 10, 15, 20]`

* **Example**:
<CodeBlock language="js">
  {`padding: [5, 10, 5, 10]`}
</CodeBlock>
</Prop>

<Prop name="transformMatrix" type="array" defaultValue="[]">
  A 6-element affine transformation matrix `[m11, m12, m21, m22, dx, dy]`.
This allows for advanced 2D transformations including translation, scaling, rotation, and shearing.

* **Matrix Structure**:
`[ ScaleX, SkewY, SkewX, ScaleY, TranslateX, TranslateY ]`

* **Example**:
<CodeBlock language="javascript">
  {`// Skew transformation
transformMatrix: [1, 0.5, 0, 1, 0, 0]`}
</CodeBlock>
</Prop>

# Tooltip Options

Options for displaying tooltips when hovering over the element.

<Prop name="tooltipText" type="string" defaultValue="">
  Text to display in the tooltip. If empty, no tooltip is shown.
</Prop>

<Prop name="tooltipTitle" type="string" defaultValue="">
  Title text displayed in bold above the tooltip text.
</Prop>

<Prop name="tooltipIcon" type="string" defaultValue="none">
  Icon to display next to the tooltip title.
* **Valid Values**:
  - `"none"`
  - `"info"`
  - `"warning"`
  - `"error"`
</Prop>

<Prop name="tooltipBalloon" type="boolean" defaultValue="false">
  If `true`, the tooltip is displayed in a cartoon-balloon style.
</Prop>

<Prop name="tooltipMaxWidth" type="number" defaultValue="1000">
  Maximum width of the tooltip in pixels. Text exceeding this width will wrap to multiple lines.
</Prop>

<Prop name="tooltipMaxHeight" type="number" defaultValue="1000">
  Maximum height of the tooltip in pixels. This is a hint for the tooltip layout, though actual height is determined by content.
</Prop>

# Mouse Actions

Events that trigger JavaScript functions in response to user interaction.

<Prop name="onLeftMouseUp, onLeftMouseDown, onLeftDoubleClick" type="function">
  Events triggered by the left mouse button.
  - `onLeftMouseUp`: Button released
  - `onLeftMouseDown`: Button pressed
  - `onLeftDoubleClick`: Button double-clicked

  **Example**:
  <CodeBlock language="javascript">
    {`onLeftMouseUp: function() { console.log("Left click!"); }`}
  </CodeBlock>
</Prop>

<Prop name="onRightMouseUp, onRightMouseDown, onRightDoubleClick" type="function">
  Events triggered by the right mouse button.
  - `onRightMouseUp`: Button released
  - `onRightMouseDown`: Button pressed
  - `onRightDoubleClick`: Button double-clicked

  **Example**:
  <CodeBlock language="javascript">
    {`onRightMouseUp: function() { console.log("Right click!"); }`}
  </CodeBlock>
</Prop>

<Prop name="onMiddleMouseUp, onMiddleMouseDown, onMiddleDoubleClick" type="function">
  Events triggered by the middle mouse button (scroll wheel click).
  - `onMiddleMouseUp`: Button released
  - `onMiddleMouseDown`: Button pressed
  - `onMiddleDoubleClick`: Button double-clicked

  **Example**:
  <CodeBlock language="javascript">
    {`onMiddleMouseUp: function() { console.log("Middle click!"); }`}
  </CodeBlock>
</Prop>

<Prop name="onX1MouseUp, onX1MouseDown, onX1DoubleClick" type="function">
  Events triggered by the X1 mouse button (usually "Back").

  **Example**:
  <CodeBlock language="javascript">
    {`onX1MouseUp: function() { console.log("Back button clicked"); }`}
  </CodeBlock>
</Prop>

<Prop name="onX2MouseUp, onX2MouseDown, onX2DoubleClick" type="function">
  Events triggered by the X2 mouse button (usually "Forward").

  **Example**:
  <CodeBlock language="javascript">
    {`onX2MouseUp: function() { console.log("Forward button clicked"); }`}
  </CodeBlock>
</Prop>

<Prop name="onMouseOver, onMouseLeave" type="function">
  Events triggered by mouse movement.
  - `onMouseOver`: Cursor enters the element area
  - `onMouseLeave`: Cursor leaves the element area

  **Example**:
  <CodeBlock language="javascript">
    {`onMouseOver: function() { console.log("Mouse entered!"); }`}
  </CodeBlock>
</Prop>

<Prop name="onScrollUp, onScrollDown" type="function">
  Events triggered by vertical scrolling.

  **Example**:
  <CodeBlock language="javascript">
    {`onScrollUp: function() { console.log("Scrolled up"); }`}
  </CodeBlock>
</Prop>

<Prop name="onScrollLeft, onScrollRight" type="function">
  Events triggered by horizontal scrolling.

  **Example**:
  <CodeBlock language="javascript">
    {`onScrollLeft: function() { console.log("Scrolled left"); }`}
  </CodeBlock>
</Prop>
