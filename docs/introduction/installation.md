# Installation
Learn how to install and set up Novadesk on your Windows system. This guide walks you through installing Novadesk on Windows.

#### Table of Contents
[[toc]]

## System Requirements

- Windows 7 or later (Windows 10/11 recommended)
- At least 20 MB of free disk space

## Novadesk Installation Guide

### Step 1: Download the Installer
Visit the official website at [https://novadesk.pages.dev/](https://novadesk.pages.dev/) and download the latest installer for Windows.

### Step 2: Run the Setup
Locate the installer (named `Novadesk_Setup_vX.X.X.X_Beta.exe`) and double-click it to launch the setup wizard.

![Installer File](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/1.png?raw=true)

### Step 3: Welcome Screen
Close other applications before continuing so system files can update smoothly.

* Click **Next >** to continue.

![Welcome Screen](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/2.png?raw=true)

### Step 4: License Agreement
Review the GNU General Public License terms.

* Click **I Agree** to accept the terms and proceed.

![License Agreement](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/3.png?raw=true)

### Step 5: Choose Install Location
Select the folder where Novadesk will be installed.

* **Default Location:** `C:\Program Files\Novadesk\`
* Use **Browse...** to pick a different folder.
* Click **Next >** to continue.

![Install Location](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/4.png?raw=true)

### Step 6: Choose Components
Check the components you want to install (all recommended options are selected by default):

* **Novadesk Core** – main program files
* **Add to PATH** – enable running Novadesk from the command line
* **Desktop Shortcut** – adds a shortcut to your desktop
* **Run on Startup** – launches Novadesk automatically

Click **Install** to begin copying files.

![Choose Components](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/5.png?raw=true)

### Step 7: Complete Installation
When the file copy finishes, you can keep **Run Novadesk** checked to launch immediately.

* Click **Finish** to close the wizard.

![Finish Setup](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/6.png?raw=true)
