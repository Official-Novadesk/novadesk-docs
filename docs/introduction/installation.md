
# Installation
Learn how to install and set up Novadesk on your Windows system.This guide will walk you through the installation process for Novadesk on Windows systems.

## `System Requirements`

- Windows 7 or later (Windows 10/11 recommended)
- At least 20 MB of free disk space

# Novadesk Installation Guide

This guide covers the installation process for the latest version of Novadesk on Windows.

### Step 1: Download the Installer
Visit the official website at [https://novadesk.pages.dev/](https://novadesk.pages.dev/) and download the latest installer for Windows.

### Step 2: Run the Setup
Locate the installation file in your downloads folder. The filename will appear as `Novadesk_Setup_vX.X.X.X_Beta.exe` (where `X` represents the current version number).

Double-click the file to launch the setup wizard.

![Installer File](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/1.png?raw=true)

### Step 3: Welcome Screen
The setup wizard will launch. It is recommended to close other applications before continuing to ensure system files can be updated without a reboot.

* Click **Next >** to continue.

![Welcome Screen](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/2.png?raw=true)
### Step 4: License Agreement
Review the license terms (GNU General Public License).

* Click **I Agree** to accept the terms and proceed.

![License Agreement](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/3.png?raw=true)

### Step 5: Choose Install Location
Select the folder where you want to install Novadesk.

* **Default Location:** `C:\Program Files\Novadesk\`
* If you wish to install it elsewhere, click **Browse...** to select a different folder.
* Click **Next >** to continue.

![Install Location](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/4.png?raw=true)

### Step 6: Choose Components
Select the features you want to install. By default, all recommended options are checked:

* **Novadesk Core:** The main program files.
* **Add to PATH:** Allows you to run Novadesk from the command line.
* **Desktop Shortcut:** Adds an icon to your desktop for easy access.
* **Run on Startup:** Automatically launches Novadesk when you turn on your computer.

Click **Install** to begin the installation.

![Choose Components](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/5.png?raw=true)

### Step 7: Complete Installation
Once the files are copied, the installation is complete. You can leave the "Run Novadesk" box checked to launch the application immediately.

* Click **Finish** to close the setup wizard.

![Finish Setup](https://github.com/Official-Novadesk/novadesk-assets/blob/master/docs/installation/6.png?raw=true)
